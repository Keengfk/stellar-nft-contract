# Contributing to Stellar NFT Contract

Thanks for considering contributing! This project is part of the Stellar Wave Program on Drips.

## How to Contribute

1. Check the [Issues](https://github.com/Keengfk/stellar-nft-contract/issues) tab for open tasks
2. Comment on an issue to claim it
3. Fork the repo and create a branch for your work
4. Make your changes
5. Test locally with:
```bash
   stellar contract build
   cargo test
```
6. Submit a pull request with a clear description

## Code Style

- Follow standard Rust formatting (`cargo fmt`)
- Keep functions focused and well-documented
- Add tests for new functionality

## Questions?

Open an issue or reach out via the repo discussions.
